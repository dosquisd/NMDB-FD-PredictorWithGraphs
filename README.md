# NMDB ForbushDecrease Predictor With Graphs

This is the continuation of this repository: [dosquisd/NMDB-FD-PredictorWithStats](https://github.com/dosquisd/NMDB-FD-PredictorWithStats/tree/main)

## Installation

To install all dependencies, it is recommended to use [uv](https://docs.astral.sh/uv/), although you can also use `pip` directly. Run the following commands based on your preference:

* **Using `uv`**

    ```console
    uv sync
    ```

* **Using `pip`**

    ```console
    # Create and activate a virtual environment
    python -m venv .venv

    # Unix-like systems
    source .venv/bin/activate
    # Windows
    .\.venv\Scripts\activate

    # Install dependencies in editable mode
    pip install -e .
    ```
